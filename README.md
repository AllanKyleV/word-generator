# Word Document Generator (Browser-based)

This project demonstrates how to **generate a Microsoft Word (.docx) file in the browser** using **HTML, CSS, and JavaScript only**, without a backend.

The user fills out a form, clicks a button, and a `.docx` file is generated and downloaded with the provided data inserted into a Word template.
